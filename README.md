# Online Purchase Intent Prediction

## Project Overview
This project develops a machine learning model to predict whether an online shopping session will result in a purchase. The analysis uses behavioural browsing data from an e-commerce platform and follows an end-to-end machine learning workflow including exploratory analysis, preprocessing, model comparison, hyperparameter tuning, and evaluation.

The final model is a Gradient Boosting classifier with a test ROC-AUC of approximately 0.928.
